# E-Commerce Website

A full-stack e-commerce website built with Next.js, React, TypeScript, and Tailwind CSS. Includes an admin panel for managing products and orders.

## Features

- Product listing and details
- Shopping cart with quantity management
- Admin panel for adding products
- Responsive design with Tailwind CSS
- API routes for product management

## Pages

- `/` - Home page with featured products
- `/products` - All products listing
- `/cart` - Shopping cart
- `/admin` - Admin dashboard for managing products

## API

- `GET /api/products` - Get all products
- `POST /api/products` - Add a new product (requires name and price)

## Getting Started

1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Admin Panel

Access the admin panel at [http://localhost:3000/admin](http://localhost:3000/admin) to add new products.
